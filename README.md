## 🚀 Featured Project

### [Solo Gadget Mart & Live Admin Suite](https://github.com/SahalMcn/solo-gadget-mart)
A complete e-commerce ecosystem built from the ground up. 
- **The Web Store:** Next.js 14+ storefront with Razorpay integration and Edge API routes.
- **The Mobile Command Center:** A highly customized Flutter application designed for the store owner.
- **The Magic:** Features a distributed real-time architecture where Next.js Node APIs securely trigger native Firebase Push Notifications (FCM) to the Admin's Android device, while Supabase WebSockets stream live revenue data directly into the BLoC-managed state without a single manual refresh.
